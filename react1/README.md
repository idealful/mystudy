## (실습) create-react-app

### npx

: eXecute NPm package binaries

```cmd
# 사용법
$ npx create-react-app <your-project-name>
```

```cmd
# 실제 사용 예제
$ npx create-react-app my-app
```

### 경로 변경 (change directory)

```cmd
$ cd my-app
```

### node_modules 폴더가 없고 package.json 파일이 있을 때

```cmd
$ npm i
```

### 애플리케이션 실행

```cmd
$ npm start
```

### 빌드 후 실행

최초 1회

```cmd
$npm i -g serve
```

```cmd
$npm run build
$serve -s build
```
